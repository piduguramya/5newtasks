Topic Description
HTML it is a standard markup language used to create webpages which acts as a structure to the website. consists of series of elements.
Markup set of rules which used to used to structure the content,images & layout.
Hypertext it is a text dispalyed on the computer or any other device where the text navigates(hyperlinks) us to the next plain which user wants.
History introduced by Tim Berners Lee
HTML 1-(1991)
HTML 5-(2014)
types of tags 1. container/pair tags(have both opening and closing tags) EX:< p >< /p >
2. empty/void/self closing tags(have only opening tag) EX:< img src="">
element it contains a start tag some content and an end tag EX:< P >IAM RAMYA< /P >
Nested elements it has parent, child and sometimes grandchild tags
